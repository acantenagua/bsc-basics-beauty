# 📋 BSC Basics Beauty — Website Renewal Plan

**Project:** Vernieuwing website BSC Basics Beauty  
**Timeline:** 2-4 weken  
**Budget:** €0 (DIY, alleen tijd)  
**Platform:** HTML/CSS + JavaScript (via jouwweb.nl hosting)  
**Domain:** basicisandbrows.nl  

---

## 🎯 Project Overview

**Doel:** Professionele beauty clinic website met:
- Alle diensten showcased (brows, lashes, PMU, facials, plasma, waxing, fillers, body treatments)
- Lead generation via contact form + Treatwell link
- Consultation-first model (contact → advies → afspraak)
- Luxe-accessible vibe (warm, editorial, professioneel)

**USP:** Fullservice beauty care waar klanten gehoord worden, eerlijk advies, extra waarde.

## 🎨 FASE 2: DESIGN SYSTEM & VISUEEL (Week 1-2)

### Goal
Visuele richtlijnen klaar, zodat HTML/CSS consistent wordt.

### Deliverables

#### 2.1 Kleurenpalet
```
Primary:   #F5E6D3 (Warm beige)
Accent:    #E8A0A0 (Zacht roze)
Dark:      #3D2817 (Warm licht bruin)
White:     #FFFFFF (Wit)
Gray:      #F9F7F4 (Off-white/cream)
```
- [ ] Kleurencombinaties testen (contrast checks)
- [ ] Variaties bepalen (light, dark, hover states)

#### 2.2 Typografie
- [ ] Headline font: **Didot** (editorial, luxe)
  - H1: 48px, bold
  - H2: 36px, bold
  - H3: 24px, regular
- [ ] Body font: **Inter** of **Lato** (warm, schoon)
  - Body: 16px, regular
  - Small: 14px, regular
- [ ] Letter-spacing: +0.5px (warm, Editorial feel)

#### 2.3 Spacing & Grid
- [ ] Basis spacing unit: **16px**
  - Padding: 16px, 24px, 32px, 48px
  - Margins: 24px, 48px, 64px
- [ ] Gulden snede: Image/text ratio 1.618
- [ ] Max-width: 1200px (desktop)

#### 2.4 Component Design
- [ ] CTA buttons: roze/beige, hover states
- [ ] Cards: behandelingen display (the Brow Spot grid inspiratie)
- [ ] Hero section: image + overlay + text
- [ ] Reviews card: quote + name + rating
- [ ] Form inputs: warm styling
- [ ] Navigation: simpel, warm

#### 2.5 Mockups (Rough)
